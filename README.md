# 项目简介
本项目致力于帮助大家快速入门鸿蒙系统应用开发。

鸿蒙系统于2019年8月正式发布，最初是为了应对它国对华为的技术禁令而开发的。它的目标是打破传统操作系统的设备限制。

在万物智联时代重要机遇期，鸿蒙系统结合移动生态发展的趋势，提出了三大技术理念：
一次开发，多端部署；可分可合，自由流转；统一生态，系统智能。

综上，学习鸿蒙应用开发是个不错的选择，最起码可以增加自己的就业机会。

目前学习鸿蒙比较好的方式就是查阅[【官方文档】](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides-V5/quick-start-V5?catalogVersion=V5)，但是官方文档太过详细，对于新手来说不是很好快速入门，所以我创建了本项目，挑选了一些比较常用的功能，依据自己的理解进行了一些讲解，并录制了相应的[【视频】](https://v.douyin.com/Yydck7EsR5g)，以便帮助大家快速入门鸿蒙应用开发，赶上这波机会。

# 学习目标

掌握鸿蒙应用开发的主要内容，包括但不限于样式布局、组件传参、全局状态管理、路由跳转、网络接口请求等等。
