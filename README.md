# 项目简介
鸿蒙系统于2019年8月正式发布，最初是为了应对美国对华为的技术禁令而开发的。它的目标是打破传统操作系统的设备限制，创建一个跨设备、跨平台的生态系统，涵盖智能手机、智能家居、可穿戴设备、车载系统等多个领域。

# 学习目标
掌握鸿蒙应用开发的主要内容，包括但不限于样式布局、组件传参、全局状态管理、路由条状、网络接口请求等等。
